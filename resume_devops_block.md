# Блок для резюме — DevOps

## Кратко в “О себе”

Дополнительно изучаю DevOps-направление: Linux/Debian, терминал, SSH, Git/GitHub, Docker, Docker Compose, Nginx, основы сетей и CI/CD. Практикуюсь на учебных проектах: деплой сайта на VPS, запуск приложения через Docker Compose и настройка автодеплоя через GitHub Actions.

## Навыки

Linux, Debian, Bash, SSH, Git, GitHub, Docker, Docker Compose, Nginx, CI/CD, GitHub Actions, VPS, основы сетей, HTTP/HTTPS, DNS, systemctl, journalctl.

## Как описать проекты

Учебные DevOps-проекты:
— деплой статического сайта на Linux/VPS через Nginx;
— запуск web-приложения и PostgreSQL через Docker Compose;
— настройка автоматического деплоя через GitHub Actions и SSH.
